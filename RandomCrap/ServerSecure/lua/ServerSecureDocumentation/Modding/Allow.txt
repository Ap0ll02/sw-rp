// Allow

SS.Allow.PlayerAllow(Player, ID, Activity, Bool) (Allow or disallow a player to do an activity, ID is a unique ID)
SS.Allow.PlayerGetAllowed(Player, Activity, Bool) (Returns if a player is allowed to do an activity)

// Activities

Spawn (Is a player allowed to spawn an object)
Tool (Can a player use the toolgun)
Noclip (Can a player use noclip)
Death (Can a player die, if not the player will respawn where the player died)
Commands (Can a player use commands)
Chat (Can a player chat)
Physgun (Can player use the physgun)
Weapons (Can player spawn with weapons)
